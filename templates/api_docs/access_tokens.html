Get Your Access Tokens

OAuth 2.0/Basic Auth

GET

/auth/api/generate-authtoken/

List all the your created access tokens

Response

status

Integer giving the status of response

The following code uses cURL to demonstrate how you can get all your created access token by OAuth 2.0 Auth (Basic Auth can also be used):

get access token

bash

curl --request GET \

--url '{{BASE_URL}}auth/api/generate-authtoken/' \

--user "Authorization: Token YOUR_ACCESS_TOKEN"

Generate Access Token

You can get your access token by 2 methods:

  1. From developer settings section in your dashboard
  2. By sending API request as shown below

Get Access Token by sending API request

Basic Auth

POST

/auth/api/generate-authtoken/

The following code uses cURL to demonstrate how you can get your access token by send an API request:

Generate Access Token

bash

curl --request POST \

--url '{{BASE_URL}}auth/api/generate-authtoken/' \

--user 'YOUR_EMAIL:YOUR_PASSWORD'

Sample Responses

json

{

"token": "330bd7af.....718bd5",

"error": null

}

Delete Access Token

OAuth 2.0

DELETE

/auth/api/delete-authtoken/<id>

The following code uses cURL to demonstrate how you can delete a access token by send an API request:

Delete Access Token

bash

curl --request DELETE \

--url '{{BASE_URL}}auth/api/delete-authtoken/12' \

--user "Authorization: Token YOUR_ACCESS_TOKEN"

Sample Responses

json

{

"status": 204,

"date": null

"message": "deleted successfully"

}
