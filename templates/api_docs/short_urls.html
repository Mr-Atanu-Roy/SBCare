OAuth 2.0

GET

Create Short URL

OAuth 2.0

POST

{{BASE_URL}}r/api/short-url/

Create a new short urls

The following code uses cURL to demonstrate how you can create a new short url:

create a short url

bash

curl --request POST \

--url '{{BASE_URL}}r/api/short-url/' \

--user "Authorization: Token YOUR_ACCESS_TOKEN" \

--data '{"original_url": "YOUR_LONG_URL", "title": "new url"}'

Sample response

json

{

"status": 201,

"data": {

"id": 2,

"title": "new url",

"original_url": "{{BASE_URL}}auth/dashboard/profile/",

"short_url": "{{BASE_URL}}r/ZCdccfJT",

"source": "api-service",

"created_at": "2023-10-11T19:54:16.184745+05:50"

}

"message": "URL created successfully"

}

Update a Short URL

OAuth 2.0

PUT

{{BASE_URL}}r/api/short-url/<id>

Update an existing short url corresponding to gievn id. Only title can be updated

The following code uses cURL to demonstrate how you can update one of your existing short url:

update short url

bash

curl --request PUT \

--url '{{BASE_URL}}r/api/short-url/10' \

--user "Authorization: Token YOUR_ACCESS_TOKEN" \

--data '{"title": "new title"}'

Sample response

json

{

"status": 202,

"data": {

"id": 10,

"title": "new title",

"original_url": "{{BASE_URL}}auth/dashboard/profile/",

"short_url": "{{BASE_URL}}r/ZCdccfJT",

"source": "api-service",

"created_at": "2023-10-11T19:54:16.184745+05:50"

}

"message": "URL created successfully"

}

Delete A Short URL

OAuth 2.0

DELETE

{{BASE_URL}}r/api/short-url/<id>

Delete one of your created short url

The following code uses cURL to demonstrate how you can delete a short url:

Delete A short URL

bash

curl --request DELETE \

--url '{{BASE_URL}}r/api/short-url/12' \

--user "Authorization: Token YOUR_ACCESS_TOKEN"

Sample Responses

json

{

"status": 204,

"date": null

"message": "deleted successfully"

}
